# Stock Analysis with Pandas and Scikit-Learn

This project fetches stock data for AAPL from 2010 to today and uses 5 different models to predict future prices

Models used are:
* Linear Regression
* Ridge
* Lasso
* Polynomial 2
* Polynomial 3
